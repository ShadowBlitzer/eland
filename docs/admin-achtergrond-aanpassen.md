###Achtergrond aanpassen

Begin in je tekst-editor naar keuze een nieuw bestand 

```css
body {
    background-color: #ffffef;
}
```

Rood

```css
body {
    background-color: #fff0f0;
}
}
```
Groen

```css
body {
    background-color: #f8fff9;
}
```

Blauw

```css
body {
    background-color: #f3f8ff;
}
```
